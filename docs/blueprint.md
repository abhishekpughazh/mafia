# **App Name**: Mafia Moderator

## Core Features:

- Game Setup & Player Management: Configure game rules (number of players, game mode, narration mode, role reveal mode), add/edit player names, and randomize seating order with a clear, mobile-first UI.
- Role Assignment & Private Reveal: Automated role distribution and a privacy-focused UI for secret, tap-and-hold role revelation to individual players, with warnings against accidental exposure.
- Dynamic Game Narration: Utilizes generative AI to create engaging, context-aware narration for Night and Day phases, announcements of eliminations, and general game guidance, adapting to the current game state.
- Night Action Management: Dedicated, private screens for specific roles (Mafia, Doctor, Detective) to perform secret night actions with clear instructions and confirmation steps.
- Voting & Elimination System: Facilitates in-person voting with options for host-entered or pass-the-phone methods, displaying vote counts, handling ties, and dramatically announcing eliminations.
- Real-time Game State Tracking: Manages and updates the ongoing game state, including alive/dead player status, role information, day/night cycles, and victory conditions to ensure smooth gameplay progression.
- Game Over & Summary Display: Presents the final game outcome, announcing the winning team, a summary of all player roles, and who survived, with options to play again or return home.

## Style Guidelines:

- Dark color scheme to evoke a mysterious and dramatic atmosphere, suitable for dimly lit rooms.
- Background color: Deep, desaturated navy blue (#16181A) providing a cinematic canvas for high-contrast elements.
- Primary color: Muted yet rich purple (#9970C2) for interactive elements, primary text, and accents, symbolizing mystery and elegance.
- Accent color: Dramatic blood red (#E02626) to highlight critical information, calls to action, and convey urgency or danger, offering strong contrast.
- Headline font: 'Playfair' (serif) for bold, dramatic, and elegant titles, reflecting a cinematic feel.
- Body font: 'Inter' (sans-serif) for clean, highly readable body text and functional UI elements, ensuring clarity from a distance.
- Use symbolic icons such as a moon, mask, eye, gavel, skull, town, vote, and shield to visually represent game elements and actions.
- Mobile-first, step-by-step navigation flow. Prioritize large, thumb-friendly buttons and readable text, minimal on-screen clutter, and clear instructions for a smooth 'pass-the-phone' experience.
- Subtle and fast transitions between screens, avoiding long or distracting animations to maintain flow, while providing delicate visual cues like slight glows or shadows on cards.
